Detecting the Gender of the Person Using Image by Opene Conputer Vision and Convolutional Neural Network

-> Usage 
# Step 1 : Go to command prompt and set working directory where the Gender_Task.py file is stored
# Step 2 : Execute the following command to detect from image: python gender_age.py 
# Step 3 : Select The image of the person with face or else the Output will show you the alert message to put the correct image with human face
# Step 4 : Output will show you the image of the person with his gender displaying as the rectangular box on his face. The image can be saved as savedImage in the same root directory.
